Roblox studio; a 3d game engine used to make roblox games.

Types of scripts

There are 3 types of scripts.

  • LocalScripts: Calculated by the client. Used for UI, Visual Effects, Playerinput, etc...

  • Script: A script calculated by the server. Used for things that are visible to all clients. Example: moving objects.

  • ModuleScript: A type of script that every Script can use. ModuleScripts are often used in things that happen repeatedly. Such as a camerashake.
