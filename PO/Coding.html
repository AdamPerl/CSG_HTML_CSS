Types of scripts

There are 3 types of scripts.

  • LocalScripts: Calculated by the client. Used for UI, Visual Effects, Playerinput, etc...

  • Script: A script calculated by the server. Used for things that are visible to all clients. Example: moving objects.

  • ModuleScript: A type of script that every Script can use. ModuleScripts are often used in things that happen repeatedly. Such as a camerashake.

Modulescripts

ModuleScripts are super usefull. They can be used to create or inprove functions, serverwide. All you need to do is require the script. (local Module = require(...) An example is Mockparts. This is a commonly used script, which you can find on the internet. This modulescript is usefull when "creating" new parts. Currently, creating new instances is wery laggy, precicly when you create tons. Mockparts improves this. Instead of creating new instances, the script generates instances slowly far away in the world. When needed, it moves the parts to the orientation and position specified. Doing so, you can create tons of bullets for weapons without lagg!
